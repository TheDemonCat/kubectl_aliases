## Linux

Запускаем файл: 

```
bash aliases_add.sh
``` 

## Windows

Настройка doskey для сокращения команд kubectl для win10

1. При необходимости правим алиасы в файле aliasesfile.doskey
2. запускаем исполняемый файл под администратором

```cmd
aliases_add.cmd
```
